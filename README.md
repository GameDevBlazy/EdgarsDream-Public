Edgar’s Dream – Phantom Dust Modding Toolkit

Edgar’s Dream is a standalone Phantom Dust modding toolkit for Windows. It provides stage editing, object scanning, texture and audio replacement, arsenal management, collision analysis, campaign and mission adjustments, and early support for map and skill workflows. This public release includes compiled tools only.

Installation

Download the latest release from the Releases tab.

Make sure Phantom Dust (PDUWP) is fully downloaded on your system.

For best results, use a dumped copy of the game.

Refer to the community wiki for sideloading and setup information:
https://phantomdust.miraheze.org/wiki/Help:Sideloading

Usage

Launch the program and follow the on-screen modules.
Most features are in active development and may change between updates.
